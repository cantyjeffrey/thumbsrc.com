thumbsrc

the free source of website thumbnails


Usage

Basic: http://www.thumbsrc.com/[url]

Example: http://www.thumbsrc.com/www.example.com


With Size: http://www.thumbsrc.com/[size]/[url]

Example: http://www.thumbsrc.com/640/www.example.com


Supported Sizes: 80, 160, 320, 480, 640
